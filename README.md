## develop

```shell
# start server
> deno task serve
# type checking
> deno task check
# run test
> deno task test
# run test (output coverage)
> deno task test:coverage
# run format
> deno fmt
# check format
> deno task test:fmt
# run lint
> deno lint
# check lint
> deno task test:lint
```
